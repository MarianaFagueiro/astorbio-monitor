# AstorBio Monitor

Painel interativo para acompanhar pesquisas biomédicas em ambiente orbital.

## Como abrir

Abra o arquivo:

`outputs/astorbio-monitor-ux.html`

O protótipo é um HTML autocontido, então não precisa instalar dependências.

## O que o painel mostra

- Saúde geral da missão.
- Saúde por doença cadastrada.
- Pesquisas, laboratórios/equipamentos e eventos.
- Cadastro guiado com exemplos, ajuda por campo e sugestão automática.
- Sincronização experimental com APIs públicas.
