Como as informações são preenchidas ?

O painel começa com dados de exemplo, mas você pode cadastrar itens manualmente ou sincronizar APIs públicas.

1Escolha a fonte

Use dados de exemplo, cadastro manual ou o botão de sincronização.

Entrada: exemplo, formulário ou API
2Preencha os campos

Pesquisas recebem doença, complexidade e progresso base de 0 a 10. Eventos recebem doença cadastrada e tipo.

Campos: 0-10, seleção e texto
3O programa calcula

O sistema transforma os dados em notas de pesquisa, laboratório e alerta. O impacto do evento é automático.

Saída: notas intermediárias
4Leia a decisão

A nota final gera status, resumo em português claro e recomendações de prioridade.

Resultado: pronto, atenção ou crítico

Mini tutorial: como mexer no site

Siga esta ordem para entender a tela sem precisar conhecer as métricas técnicas.

1. Veja como os dados entram

A primeira seção da página mostra se os dados vêm de exemplo, cadastro manual ou APIs.

2. Escolha uma ação no topo

Use “Exemplo” para testar, “Cadastrar” para inserir dados, ou “Sincronizar APIs” para buscar fontes públicas.

3. Leia a saúde geral

A nota grande resume a missão inteira. Ela serve para saber se o sistema todo está bem.

4. Compare por doença

Em “Saúde por doença”, veja o IPO específico de cada doença cadastrada.

5. Registre acontecimentos

Em “Eventos e alertas”, selecione a doença cadastrada e o tipo do evento. O programa calcula o impacto sozinho.

6. Alterne Simples/Técnico

Use “Simples” para entender rápido e “Técnico” para ver fonte, pesos e detalhes do cálculo.
